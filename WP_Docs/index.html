# WTDR.Pack 是什么？

WTDR.Pack 为 WTDR 内的组件，为了更好的发展将其分离出来，不用依赖 WTDR 可以独立调用
WTDR.Pack 可以在部署过程中应用相应的包(Pack)，可以释放相应的程序、文件或者应用相应设置，可以理解为是一个运行任务包，应用他就是执行你配置好的任务

注意

该文档仅适用于 1.6.0 或更高版本的 WTDR.Pack！

# 用途

可控制系统部署时要预装的软件
第三方社区系统则方便用户对系统镜像进行 DIY
还有无限可能 xD
