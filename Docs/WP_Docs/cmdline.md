# 命令行参数

结合文档使用以下参数，你可以将 WTDR.Pack 应用到很多项目中   

| 参数               | 用途               |
| -------------------- | -------------------- |
| /ApplyConfig         | 按照当前配置应用 |
| /Office              | 寻找并应用 Office |
| /ApplyTool           | 直接打开应用单包 GUI |
| /p=配置文件&#124;包文件夹 | 按照指定配置应用 |
| /s=包路径         | 应用单个包      |
| /Deploy              | 当前为部署环境 |
| /Desktop             | 当前为桌面环境 |
